ERT300 String Tension Converter

A web app for tennis players and stringers. Converts ERT300 Dynamic Tension (DT) readings to kg/lbs, tracks string tension over time, and maintains a historical log of stringing jobs.

How it works

My Rackets

Your equipment registry. Each entry is a physical racket frame you own. Define the name (e.g. Racket 1), model, and head size. Stringing data flows in automatically from Stringing Jobs — no double entry.

Stringing Jobs

The single source of truth for all stringing data. Log every stringing job with model, string, machine tension, baseline DT, date, and notes. Assign each job to a racket from your registry using the dropdown — this auto-fills the model and head size and sets the baseline on that racket card.

DT history

After stringing, measure DT periodically and add follow-up readings directly on the racket card. The baseline (from the stringing job) is read-only. Restringing is recommended when tension loss reaches 20%.

Converter

Standalone DT → kg/lbs converter. Use the +/− stepper to enter a DT value, select the head size, and read the result. The DT zones legend highlights the active playing style zone.

Chart

In the Stringing Jobs tab, tap Show chart to see a scatter plot of machine tension vs baseline DT for all your jobs. Filter by racket model to see the trend for a specific frame — useful for predicting what machine tension to use to hit a target DT.

Cloud sync

Tap Sign in with Google to sync all data across devices via Firebase. A green dot in the My Rackets tab confirms sync is active. Without signing in, data is saved locally in the browser.

Settings (⚙)

Tap the gear icon next to the sign-in button to access backup and restore options:

  • Export data (JSON) — saves a full backup of all rackets, stringing jobs, DT history, and sort order to a JSON file.
  • Import data (JSON) — restores data from a previously exported JSON file. Merges with existing data. If you sign in after importing, cloud data will take precedence over local data.
  • Restore from JSON (clean) — deletes all existing data (locally and from Firestore if signed in), then imports the JSON file fresh. No conflicts. Use this when migrating to a new device or account. This cannot be undone.

Adding to iPhone home screen

Open the app URL in Safari, tap the Share button, then tap Add to Home Screen. The app will open full-screen like a native app. After updating the app on GitHub, delete and re-add from the home screen to pick up changes.
